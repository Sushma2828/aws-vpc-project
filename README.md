# # AWS VPC Project

## Project Overview
This project demonstrates creation of a custom VPC in AWS.

## Components Created
- VPC
- Public Subnet
- Private Subnet
- Internet Gateway
- Route Table
- EC2 Instance

## Project Steps
1. Created VPC
2. Created Subnets
3. Attached Internet Gateway
4. Configured Route Table
5. Launched EC2 instance
6. Hosted web server

## Outcome
Successfully deployed a web server inside a custom VPC.
